# Clock Application

**Version:** 1.0  
**Language:** Python 3.14.1  
**GUI Library:** Tkinter

## Overview
This application is a modern clock displaying the date above the time. Optional milliseconds can be shown. It features a side settings panel that slides in from the left, accessible via a hamburger menu button (â˜°). The panel automatically closes when clicking outside it.

## Features
- Large clock style time display.
- Date displayed above the time.
- Optionally shows milliseconds.
- Customizable colors for time and date.
- Animated settings panel toggled by hamburger menu.
- Click outside the panel closes it automatically.

## Installation
1. Ensure Python 3.14.1 is installed.
2. Tkinter is required (usually included with Python).
3. Place all files in the same folder:
   - `clock.pyw`
4. Run the application:
   - On Windows: double-click `clock.pyw`
   - Or from terminal: `python clock.pyw`

## Customization
- Font: Uses "Anton" for a modern bold look (install if not present).
- Window Size: Default 600x300, can be resized in the code.
- Panel Width: Default 220px, adjustable in the code.

## Notes
- Fully self-contained; no external images or libraries required.
- Best visual experience with Anton font installed.
